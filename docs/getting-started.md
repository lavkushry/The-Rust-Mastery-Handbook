# Getting Started

This page helps first-time visitors run and read the handbook quickly.

## What You Get

- Local interactive reading via mdBook
- Structured multi-part Rust learning path
- Optional PDF export for offline reading

## Fastest Path

```bash
cargo install mdbook --locked
git clone https://github.com/lavkushry/The-Rust-Mastery-Handbook.git
cd The-Rust-Mastery-Handbook
mdbook serve --open
```

## Next Steps

- Start at src/00_title_and_toc.md in the local site.
- Follow parts in order unless you are targeting a specific topic.
- Use appendices for quick reference and drills.
