# Legacy Aggregate Sources

These files are the pre-chapterization source archives for the handbook.

They were kept so the earlier monolithic drafts remain available for reference, but they are no longer the active mdBook source layout.

Active source of truth:

- `src/part-XX/` for part landing pages and chapter pages
- `src/appendices/` for appendix pages
- `src/SUMMARY.md` for navigation

If you continue editing the handbook, edit the split chapter pages rather than these archived aggregate files.
