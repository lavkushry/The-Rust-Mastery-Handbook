# Social Preview Image Spec

This document defines a recommended social preview asset for GitHub and link sharing.

## Goal

When someone shares the repository, the preview should instantly communicate:

- this is a Rust handbook
- this is systems-level and contributor-oriented
- this is a practical learning resource, not just notes

## Recommended Asset

- File name: social-preview-rust-mastery-handbook.png
- Aspect ratio: 1280 x 640
- Format: PNG
- Visual style: clean, high contrast, readable at small size

## Suggested Layout

Top-left:
- Title: The Rust Mastery Handbook
- Subtitle: From foundations to real open-source contribution

Center:
- 3 concise pillars:
  - Ownership and lifetimes
  - Async and systems patterns
  - Open-source contribution readiness

Bottom-right:
- GitHub handle or project mark: lavkushry

## Copy Constraints

- Keep text under 20 words for core headline area.
- Avoid dense paragraph text.
- Avoid marketing-heavy language.

## Accessibility

- Ensure minimum contrast ratio for key text.
- Avoid tiny typography in dense background zones.

## Manual Setup

Upload in GitHub UI:

1. Repository Settings
2. General
3. Social preview
4. Upload image
